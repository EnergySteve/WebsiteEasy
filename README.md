# Energy Landing Page Builder

A one-prompt personal brand landing page generator. Built for entrepreneurs, creators, and coaches who want a high-converting single-page site without the overwhelm.

## How to use

1. Install Claude Code: https://docs.claude.com/claude-code
2. Open your terminal and run:

```
git clone https://github.com/EnergySteve/energy-landing-page.git
cd energy-landing-page
claude
```

3. Once Claude Code opens, paste this:

```
Read PROMPT.md and follow it exactly. Start by asking me question 1.
```

4. Answer the 12 questions. Claude builds your landing page as a single index.html file.
5. Open index.html in your browser or deploy to Vercel, Netlify, or GitHub Pages.
