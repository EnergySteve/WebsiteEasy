# Energy Landing Page Builder

A one-prompt personal brand landing page generator. Built for entrepreneurs, creators, and coaches who want a high-converting single-page site without the overwhelm.

## How to use

### Step 1 — Install Claude Code (one time setup)
Go to https://docs.claude.com/claude-code and follow the install instructions for your computer.

### Step 2 — Open Claude Code
- **Mac:** Press `Cmd + Space`, type "Terminal", hit Enter. Then type `claude` and hit Enter.
- **Windows:** Press Windows key, type "Terminal" or "PowerShell", hit Enter. Then type `claude` and hit Enter.

### Step 3 — Paste this ONE message into Claude Code

```
Please do the following, in order:

1. Clone this repo into the current folder: https://github.com/EnergySteve/WebsiteEasy
2. Read the PROMPT.md file inside the repo
3. Follow the instructions in PROMPT.md exactly — start by asking me question 1
4. When I've answered all 12 questions, build my landing page as a single index.html file in the WebsiteEasy folder
5. After it's built, tell me how to open it and how to deploy it

Go.
```

### Step 4 — Answer the 12 questions

Claude asks you 12 questions about your brand, offer, and vibe. Answer in your own voice.

### Step 5 — Your site is built

Claude creates `index.html` in the WebsiteEasy folder. Double-click to open in your browser. Ask Claude "how do I deploy this?" to put it online free in 2 minutes.
