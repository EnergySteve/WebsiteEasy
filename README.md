# Easy Website Builder

A one-prompt personal brand landing page generator. Built for entrepreneurs, creators, and coaches who want a high-converting single-page site without the overwhelm.

## How to use

### Step 1 — Download the Claude desktop app
Go to https://claude.ai/download and install it on your Mac or Windows.

### Step 2 — Open the Code section
Open the Claude app. In the top left, click the **Code** tab (looks like `</>`).

### Step 3 — Paste this ONE message

In the "Describe a task or ask a question" box, paste this:

```
Please do the following, in order:

1. First, change to my Desktop folder: cd ~/Desktop
2. Clone this repo: https://github.com/EnergySteve/WebsiteEasy
3. Read the PROMPT.md file inside the WebsiteEasy folder
4. Follow the instructions in PROMPT.md exactly — start by asking me question 1
5. When I've answered all 12 questions, build my landing page as a single index.html file in the WebsiteEasy folder
6. After it's built, tell me how to open it and how to deploy it

Go.
```

Hit Enter.

### Step 4 — Answer the 12 questions
Claude asks you 12 questions about your brand, offer, and vibe. Answer in your own voice. It'll also ask permission to run commands — just say yes.

### Step 5 — Your site is built
Claude creates `index.html` on your Desktop → WebsiteEasy → index.html. Double-click to open in your browser.

### Step 6 — Put it online (optional)
Ask Claude: *"How do I deploy this to Vercel?"* — it'll walk you through it free in 2 minutes.
